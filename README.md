# Quiz App

This project was created as a **practice mini project** while learning **JavaScript**. It helped reinforce core JavaScript concepts such as DOM manipulation, event handling, arrays, functions, conditional statements, and dynamic UI updates by building a real-world interactive application.

## Features

* Multiple-choice quiz
* Score tracking
* Instant answer validation
* Highlights correct and incorrect answers
* Disables options after selection
* Next question navigation
* Play Again functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
